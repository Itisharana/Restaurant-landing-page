# Restaurant-landing-page

A clean and responsive **restaurant website** built using HTML, CSS, and Bootstrap.
This project focuses on creating a modern UI with proper layout, responsiveness, and minimal custom styling.

##  Features

* Responsive design for all screen sizes
* Clean and structured layout
* Navigation bar with centered search
* Hero section with background image
* Menu section with equal-sized cards
* Hover effects for better user experience

---

##  Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout customization
* **Bootstrap 5** – Responsive design and components
